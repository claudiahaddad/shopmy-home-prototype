# ShopMy Home Prototype

Theme-focused Home page prototype for ShopMy — personalized theme carousels, product cards, and curator social proof.

## Live demo

After GitHub Pages is enabled: `https://<username>.github.io/shopmy-home-prototype/`

## Local preview

```bash
python3 -m http.server 8765
```

Open [http://localhost:8765](http://localhost:8765)

## Structure

- `index.html` — Home with theme banners + carousels
- `theme.html` — Theme detail page
- `data.js` — Themes, products, card rendering
- `assets/` — Theme header collages and curator avatars
